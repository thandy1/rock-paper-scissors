# Rock Paper Scissors

A browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## About
An interactive Rock Paper Scissors game where the player competes against 
the computer in a best of 3 match. First to 2 wins takes the game.

## Features
- Best of 3 match system — first to 2 wins
- Animated hand shake sequence on each round
- Real time score and round tracking
- Responsive design across mobile, tablet, and desktop
- Play again and quit options

## How To Play
1. Click Rock, Paper, or Scissors to make your choice
2. Watch the hands shake and reveal the result
3. First to 2 wins takes the match
4. Hit Play Again to start a new session or Quit to end

## Tech Stack
- HTML5
- CSS3 
- JavaScript (ES6+)
